# Pembagian Tugas Kelompok - Pemrograman Lanjut

Dokumen ini berfungsi sebagai penjelasan dan klarifikasi mengenai pembagian tugas dalam pengerjaan Lembar Kerja Pemrograman Lanjut pada repositori ini.

## Mekanisme Pengerjaan

Seluruh proses pengerjaan tugas ini dilakukan secara **bersama-sama (kolaboratif dan simultan)** oleh semua anggota kelompok. Kami memanfaatkan teknologi dan alat bantu berikut untuk menunjang kolaborasi:

1.  **VS Code Live Share Extension**
    Kami menggunakan ekstensi **Live Share** pada Visual Studio Code. Melalui *tool* ini, kami dapat mengetik, melakukan *debugging*, dan menyusun baris kode secara *real-time* di dalam satu *workspace* yang sama.
2.  **Discord (Komunikasi & Koordinasi)**
    Sepanjang proses *coding*, kami terhubung melalui *voice channel* dan *screen sharing* di **Discord**. Hal ini memungkinkan kami untuk mendiskusikan logika pemrograman, menyelesaikan *bug*, dan bertukar ide secara langsung saat itu juga.

## Detail Pembagian Tugas

Karena metode pengerjaan yang bersifat kolaboratif penuh secara *live*, **kami tidak memiliki pembagian tugas spesifik per fitur atau per berkas (file)**. 

Setiap baris kode dan komponen dalam lembar kerja ini merupakan hasil diskusi, buah pikiran, dan pengerjaan bersama dari seluruh anggota kelompok secara interaktif. Oleh karena itu, seluruh kontribusi terdokumentasi secara kolektif selama sesi kolaborasi tersebut berlangsung.
